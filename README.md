# Eimeria Classification with Gemini

A Gemini-based image classification framework for identifying **Eimeria parasite species** from chicken image data.

## Overview

This project uses Google's Gemini multimodal models to classify parasite images into Eimeria species classes based on image input and text prompts. The repository is designed as a research-oriented prototype for evaluating the ability of large multimodal models to recognize and classify parasite species from microscopy or related image data.

## Features

- Image-based parasite classification with Gemini
- Prompt-based one-word class prediction
- Batch processing of image folders
- CSV export of model predictions
- Simple experimental workflow for research and evaluation

## Target Classes

The current workflow is designed for classification among the following Eimeria classes:

- ACERVULINA
- BRUNETTI
- MAXIMA
- MITIS
- NECATRIX
- PRAECOX
- TENELLA
